git commit
commit staged snapshot
commits are basically file saves in git
they save every file in the repository
all commits are logged chronologically

Git log
shows commit history for repository

git diff <id1> <id2>
shows the differences between the files

git diff
shows differences between staging area and workspace

git diff --staged
shows difference between staging area and most recent commit

git diff HEAD
shows difference between working area and most recent commit

git clone <url> 
copies repository from url to local machine

git checkout <id>
snags an old commit so you can look at it
enters detached head state
